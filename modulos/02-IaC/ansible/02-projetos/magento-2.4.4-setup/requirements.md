## Guia de instalação 

A seguir será apresentada uma lista de dependências de software e serviços para execução do Magento 2.4.4.

### Requisitos do sistema

Para a instalação e configuração bem sucedida do magento é necessário os seguintes softwares:

Composer:
* Composer 2.1

Aplicação:
* PHP 8.1

Web server:
* Apache 2.4
* nginx 1.22

Cacheamento:
* Varnish 7.0
* Elasticsearch 7.16
* OpenSearch 1.2
* AWS ElastiCache Redis 6
* AWS Elasticsearch 7.9

Banco de dados:
* MariaDB 10.4
* MySQL 8.0
* AWS Aurora (MySQL) 5.7

MQ:
* RabbitMQ 3.9
* AWS MQ 3.8.11

</br>
